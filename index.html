Weekly Reset will reset all the weekly gain stats immediately, so please make note of them before resetting!

The first weekly will start tracking the first time the page loads. It is recommended to wait until the next week before using this tool to track gains.

Likewise, new members will only start being tracked on weekly the first time they appear on this page. It is recommended to load this page whenever a new member you want to track joins the guild.
